# Hello-Tamil
Hello Tamil is an AI-powered mobile learning system for non-native children under 8 in Sri Lanka. It uses gamification, handwriting and speech feedback, and Tamil–Sinhala–English bilingual support to improve early Tamil letter recognition, writing, pronunciation, and vocabulary skills.

---

## 📌 Research Problem

Non-native Tamil-speaking children in Sri Lanka face:
- Limited early exposure to Tamil
- Traditional and non-engaging teaching methods
- Difficulty in letter recognition, writing, and pronunciation
- Lack of child-friendly, bilingual digital learning tools
- Limited ability of parents to support Tamil learning

---

## 🎯 Research Objectives

- Develop an interactive mobile application for early Tamil learning  
- Improve letter recognition, writing, pronunciation, and vocabulary  
- Integrate gamified learning approaches to increase engagement  
- Provide AI-based handwriting and speech feedback  
- Support Tamil–Sinhala–English bilingual learning  
- Evaluate learning effectiveness through real-world user testing  

---

## 🧩 System Modules

### 1. Interactive Letter Identification
- Game-based Tamil letter recognition
- Audio-visual learning support
- Rewards, stars, and progress tracking

### 2. Adaptive Tamil Writing Training
- Step-by-step stroke guidance
- Real-time predictive error detection
- Personalized learning based on writing behavior

### 3. Bilingual Word Translation
- Tamil → Sinhala / English translation
- Text and image-based input (OCR)
- Pronunciation audio playback
- Gamified word hunts and challenges

### 4. Tamil Speech Training
- Pronunciation practice using speech recognition
- Tamil phonetic-focused training
- Child-friendly feedback with animations and rewards
